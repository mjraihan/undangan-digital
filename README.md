<img width="1366" height="652" alt="image" src="https://github.com/user-attachments/assets/e3275326-33d8-4071-9713-e658058d341e" /># 💍 Undangan Digital

Website undangan pernikahan digital yang modern, responsif, dan interaktif. Dibuat menggunakan HTML, CSS, dan JavaScript dengan tampilan elegan serta animasi yang halus untuk memberikan pengalaman terbaik bagi para tamu.

## ✨ Fitur

- 🎉 Tampilan landing page yang elegan
- 📱 Responsive di desktop maupun mobile
- 🎵 Background music
- 💌 Informasi mempelai
- 📅 Detail acara
- 📍 Lokasi acara
- ⏳ Countdown menuju hari pernikahan
- 📷 Gallery foto
- 💬 Ucapan & doa dari tamu
- 🎁 Informasi hadiah (Wedding Gift)
- 🎬 Animasi halaman
- 🔒 Admin panel untuk mengelola data

## 🚀 Cara Menjalankan

Clone repository

```bash
git clone https://github.com/mjraihan/undangan-digital.git
```

Masuk ke folder project

```bash
cd undangan-digital
```

Jalankan menggunakan Live Server (VS Code) atau web server lokal lainnya.
